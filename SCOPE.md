# Project Scope - Working Project

## Goal
Working Project is a personal backend-focused project.

## In Scope
- Create & manage projects
- Create & manage project tasks
- Task status lifecycle management (planned, in progress, completed, cancelled) using enum
- Task time tracking (multiple work sessions)
- Task history (status change & activity log)
- Reason logging for task creation & updates
- Generate reports for task progress and worklog per project
- Filter task by status, date range, project
- Pagination and sorting sorting for task listing
- API documentation (OpenAPI/Swagger)
- Parameterized reports (date range, project)

## Out of Scope
- Frontend UI
- Authentication & Authorization

## Technical Scope
- REST API
- Structured JSON response
- Validation & error handling
- Relational database with ERD
